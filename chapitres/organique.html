🔗 Isomérie
Isomères de constitution
Même formule brute, squelette carboné différent ou position du groupe fonctionnel différente. Ex: propan-1-ol et propan-2-ol (C₃H₈O).
Stéréoisomères (Z/E)
Même formule développée, mais orientation spatiale différente autour d'une double liaison C=C. Z (zusammen = ensemble) : groupes prioritaires du même côté. E (entgegen = opposé) : côtés opposés.

✏️ Exercice type BAC
1
Identifier et nommer
Facile
Donner la famille fonctionnelle et le nom de : CH₃-CH₂-CH₂-OH
📋 Méthode de nomenclature IUPAC
1) Identifier le groupe fonctionnel
2) Trouver la chaîne principale la plus longue
3) Numéroter pour donner le plus petit indice au groupe fonctionnel
① Identifier le groupe fonctionnel
Présence de −OH → Alcool (suffixe : −ol)
② Compter les carbones de la chaîne principale
CH₃−CH₂−CH₂−OH → 3 carbones → racine : prop−
③ Numéroter
OH sur C1 (en partant du côté OH pour le plus petit indice)
→ propan-1-ol
✅ Nom : propan-1-ol. Formule semi-développée : CH₃CH₂CH₂OH.
2
Isomérie
Facile
CH₃COOCH₃ et CH₃CH₂COOH ont la même formule brute C₃H₆O₂. Sont-ils isomères ? Identifier leurs familles.
📋 Définition isomères de constitution
Même formule brute + squelette ou position du groupe fonctionnel différents.
① Vérifier la formule brute
CH₃COOCH₃ : 3C + 6H + 2O ✓
CH₃CH₂COOH : 3C + 6H + 2O ✓
→ Même formule brute C₃H₆O₂ → Oui, ce sont des isomères de constitution
② Identifier les familles
CH₃COOCH₃ : groupe −COO− (lien O entre deux groupes carbones) → Ester
Nom : éthanoate de méthyle (acide éthanoïque + méthanol)
CH₃CH₂COOH : groupe −COOH en bout de chaîne → Acide carboxylique
Nom : acide propanoïque
✅ Même formule brute, groupes fonctionnels différents → isomères de fonction.
