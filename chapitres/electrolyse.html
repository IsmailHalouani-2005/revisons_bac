⚡ Électrolyse

Transformation forcée d'énergie électrique en chimique. Calcul de masse déposée.

⚡ Principe de l'électrolyse

L'électrolyse est une transformation forcée non spontanée : on impose un courant électrique pour faire se produire une réaction chimique qui n'aurait pas lieu spontanément.

📐 Calculs quantitatifs
Charge électrique totale
$Q = I\Delta t \quad (\text{C} = \text{A·s})$
Lien charge – moles d'électrons
$Q = n(e^-)F \qquad F = 96\,485\,\text{C·mol}^{-1}$
Masse déposée (cathode)
$m = nM = \dfrac{I\Delta t·M}{zF}$

z = nombre d'électrons échangés par ion dans la demi-équation. Ex : Cu²⁺ + 2e⁻ → Cu : z = 2.

1
Écrire les demi-équations à l'anode (oxydation) et cathode (réduction).
2
Calculer Q = I·Δt puis n(e⁻) = Q/F.
3
Déduire n(métal) = n(e⁻)/z puis m = n·M.
🧠 Anode vs Cathode — le moyen mnémotechnique
💡
OA CA = "Oh! A Ca !" → Oxydation à l'Anode, Réduction à la Cathode. Pareil pour les piles !
📐 Formules clés
Charge électrique totale
$Q = I\Delta t \qquad Q = n(e^-)F$
Nombre de moles d'électrons
$n(e^-) = \dfrac{I\Delta t}{F} \qquad F = 96\,500\,\text{C·mol}^{-1}$
Masse déposée (électrodépôt)
$m = nM = \dfrac{I\Delta t·M}{zF} \qquad (z = \text{nb e}^- \text{échangés})$
⚠️
z = nombre d'électrons échangés dans la demi-équation. Ex: Cu²⁺ + 2e⁻ → Cu : z = 2. Ag⁺ + e⁻ → Ag : z = 1.
✏️ Exercice type BAC
1
Électrodépôt de cuivre
Moyen
Électrolyse d'une solution de CuSO₄. I = 2,0 A, Δt = 30 min. M(Cu) = 63,5 g/mol, F = 96 500 C/mol.
Demi-équation cathode : Cu²⁺ + 2e⁻ → Cu (z = 2)
Calculer la masse de cuivre déposée.
📋 Données
I = 2,0 A · Δt = 30 min = 1800 s · M(Cu) = 63,5 g·mol⁻¹ · F = 96 500 C·mol⁻¹
Demi-équation : Cu²⁺ + 2e⁻ → Cu → z = 2
① Q = I × Δt
Q = 2,0 × 1800 = 3600 C
② n(e⁻) = Q/F
n(e⁻) = 3600 / 96 500 = 3,73×10⁻² mol
③ Stœchiométrie : 1 mol Cu ← 2 mol e⁻
n(Cu) = n(e⁻)/z = 3,73×10⁻²/2 = 1,87×10⁻² mol
④ m(Cu) = n × M
m = 1,87×10⁻² × 63,5 = 1,19 g
