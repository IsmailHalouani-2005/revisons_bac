Sites donneurs/accepteurs d'électrons, flèches courbes. La chimie au niveau microscopique.

🔧 Les flèches courbes

Une flèche courbe représente le déplacement d'un doublet d'électrons. Elle part toujours de là où sont les électrons (doublet, liaison) vers là où ils vont.

⚠️
La flèche ne représente PAS le mouvement d'un atome. Elle représente le mouvement de 2 électrons. Elle va TOUJOURS du riche en électrons vers le pauvre.

Doublet non liant (N:, O:, halogens)Lacune électronique (H⁺, carbocation)
Liaison C=C (électrons π)Atome δ⁺ dans liaison polarisée
Liaison C=O (paire libre sur O)Atome chargé positivement
Carbanion R⁻Liaison polarisée vers l'atome électronégatif
Polarisation d'une liaison
Dans R−X (X électronégatif comme Cl, O, N) : X attire les électrons → C^(δ+)−X^(δ−). Le carbone est δ+ donc électrophile (accepteur). X est δ− donc nucléophile (donneur).
🔄 Mécanisme de substitution nucléophile (SN)
1
Identifier le nucléophile (donneur, ex: HO⁻, NH₃) et l'électrophile (accepteur, ex: C−Cl).
2
Flèche du doublet du nucléophile vers le C^δ+.
3
Flèche de la liaison C−X vers X (qui part avec les 2 électrons comme X⁻).
4
Écrire le produit formé et vérifier les charges.

✏️ Exercice type BAC
1
Identifier les sites
Moyen
Dans la réaction : HO⁻ + CH₃-Cl → CH₃-OH + Cl⁻
1. Identifier HO⁻ : site donneur ou accepteur ?
2. Identifier le carbone dans CH₃-Cl : donneur ou accepteur ?
3. Dessiner la flèche courbe de l'attaque.
📋 Rappel
Site donneur (nucléophile) = riche en électrons (doublets, liaisons π)
Site accepteur (électrophile) = pauvre en électrons (charge +, δ+)
① HO⁻ : donneur ou accepteur ?
HO⁻ porte des doublets non liants sur O → donneur (nucléophile)
Les ions chargés négativement sont généralement nucléophiles.
② C dans CH₃−Cl : donneur ou accepteur ?
Cl est très électronégatif → attire les e⁻ → C est appauvri → Cδ+
→ accepteur (électrophile)
③ Flèches courbes du mécanisme SN2
Flèche 1 : du doublet de HO⁻ (sur O) → vers Cδ+ de CH₃Cl
Flèche 2 : de la liaison C−Cl → vers Cl (qui part avec les 2 électrons en Cl⁻)
✅ C'est une substitution nucléophile (SN2) : HO⁻ attaque, Cl⁻ part simultanément.
