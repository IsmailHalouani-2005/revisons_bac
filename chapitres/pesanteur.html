🪂 Mouvement dans le Champ de Pesanteur

Chute libre, tir parabolique, portée et flèche. Le chapitre le plus demandé au BAC.

🌍 Pourquoi une parabole ?

Dans le champ de pesanteur, la seule force est le poids P = mg vers le bas. Pas de frottement (approximation).

Résultat de Newton : ax = 0 (horizontalement, rien ne pousse) et ay = −g (verticalement, la pesanteur tire vers le bas).

Ces deux mouvements sont indépendants et superposés. Le résultat est une parabole.

💡
Horizontalement = MRU (vitesse constante). Verticalement = MRUV (accélération g vers le bas). Ces deux mouvements se déroulent en même temps.
📐 Équations complètes du tir parabolique

Lancer depuis l'origine (x₀=0, y₀=0) avec vitesse initiale v₀ à l'angle α :

Décomposition de v₀
$v_{0x} = v_0\cos\alpha \qquad v_{0y} = v_0\sin\alpha$
Accélérations
$a_x = 0 \qquad a_y = -g = -9{,}81\,\text{m·s}^{-2}$
Vitesses
$v_x(t) = v_0\cos\alpha \qquad v_y(t) = v_0\sin\alpha - gt$
Positions
$x(t) = v_0\cos\alpha·t \qquad y(t) = -\tfrac{1}{2}gt^2 + v_0\sin\alpha·t$
Équation de la trajectoire (éliminer t)
$y = x\tan\alpha - \dfrac{gx^2}{2v_0^2\cos^2\alpha}$
🎯 Cas particulier : lancer horizontal (α = 0)

Si on lance horizontalement depuis une hauteur h (sin 0=0, cos 0=1) :

Cas α = 0
$x(t) = v_0 t \qquad y(t) = -\tfrac{1}{2}gt^2 + h$

Pour trouver quand l'objet touche le sol : y(t) = 0 → t = √(2h/g). Puis portée = x(t).

🧮 Méthode BAC
1
Décomposer v₀ en composantes x et y selon l'angle α.
2
Écrire x(t) et y(t) selon les formules ci-dessus (ou les retrouver en intégrant a).
3
Pour trouver un temps particulier : poser la condition (y = 0 pour sol, vy = 0 pour flèche...).
4
Injecter t dans l'autre équation pour trouver x ou y.
🧠 L'image mentale
💡
Lance un ballon en diagonal : horizontalement, il va à vitesse constante (pas de friction). Verticalement, la gravité le tire vers le bas. Ces deux mouvements sont INDÉPENDANTS. La trajectoire résultante est une parabole.
📐 Équations du mouvement (à connaître par cœur)
Accélération (seule force = poids)
$a_x = 0 \qquad a_y = -g$
Vitesse (primitive de a)
$v_x(t) = v_0\cos\alpha \qquad v_y(t) = v_0\sin\alpha - gt$
Position (primitive de v)
$x(t) = v_0\cos\alpha·t \qquad y(t) = -\tfrac{1}{2}gt^2 + v_0\sin\alpha·t$
Équation de la trajectoire (y en fonction de x)
$y(x) = -\dfrac{gx^2}{2v_0^2\cos^2\alpha} + x\tan\alpha$
⚠️
Si lancé horizontalement (α = 0) : sin 0 = 0 et cos 0 = 1, donc vₓ = v₀, vᵧ = −gt, x = v₀t, y = −½gt²

✏️ Exercices type BAC
1
Bille lancée horizontalement
Facile
Une bille est lancée horizontalement depuis une table (hauteur h = 1,25 m) avec v₀ = 3 m/s. g = 10 m/s².
1. Écrire x(t) et y(t) (origine = bord de la table).
2. Quand touche-t-elle le sol ?
3. Quelle est la portée horizontale ?
📋 Données
h = 1,25 m · v₀ = 3 m·s⁻¹ · α = 0° · g = 10 m·s⁻²
Origine : bord de la table → sol à y = −1,25 m
① Équations du mouvement (α = 0 : sin = 0, cos = 1)
x(t) = v₀t = 3t  |  y(t) = −½gt² = −5t²
② Temps de chute : y(t) = −1,25
−5t² = −1,25 → t² = 0,25 → t = 0,5 s
③ Portée
x = 3 × 0,5 = 1,5 m
2
Tir oblique (BAC classique)
Moyen
Un ballon est lancé avec v₀ = 15 m/s à α = 30° depuis le sol. g = 10 m/s².
1. Exprimer x(t) et y(t). (cos 30° ≈ 0,866 ; sin 30° = 0,5)
2. Calculer la hauteur maximale.
3. Calculer la portée (quand y revient à 0).
📋 Données
v₀ = 15 m·s⁻¹ · α = 30° · cos 30° = 0,866 · sin 30° = 0,5 · g = 10 m·s⁻²
① Décomposer v₀
v0x = 15×0,866 = 13 m·s⁻¹  |  v0y = 15×0,5 = 7,5 m·s⁻¹
② Équations du mouvement
x(t) = 13t  |  y(t) = −5t² + 7,5t
③ Hauteur max : vy(t) = 0
−10t + 7,5 = 0 → tflèche = 0,75 s
ymax = −5×(0,75)² + 7,5×0,75 = −2,81 + 5,625 = 2,81 m
④ Portée : y(t) = 0
t(−5t + 7,5) = 0 → t = 1,5 s (t = 0 = départ)
x = 13 × 1,5 = 19,5 m
💡 tvol = 2 × tflèche = 1,5 s ✓ (toujours vrai pour lancer depuis le sol)
