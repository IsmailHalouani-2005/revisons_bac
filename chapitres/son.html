🔊 Intensité Sonore

Décibels, niveau d'intensité sonore, atténuation. La physique du son.

🔊 Qu'est-ce que le son ?

Le son est une onde mécanique longitudinale : les molécules d'air vibrent dans la même direction que la propagation. Il faut un milieu matériel pour se propager (pas de son dans le vide).

Vitesse du son : c ≈ 340 m/s dans l'air à 20°C. Plus rapide dans les liquides (~1500 m/s) et solides.

Intensité sonore I
Puissance acoustique reçue par unité de surface. Unité : W·m⁻². I₀ = 10⁻¹² W·m⁻² = seuil d'audibilité (le plus faible son audible).
📊 Niveau d'intensité sonore L (décibels)

L'oreille humaine perçoit les sons de façon logarithmique (pas linéaire). D'où l'échelle en décibels :

Niveau sonore
$L = 10\log\!\left(\dfrac{I}{I_0}\right)\ \text{(dB)} \qquad I_0 = 10^{-12}\ \text{W·m}^{-2}$
Retrouver I
$I = I_0 ·10^{L/10}$
⚠️
Les niveaux sonores NE s'additionnent PAS ! Pour 2 sources identiques à L dB : Ltotal = L + 10·log(2) = L + 3 dB. Il faut toujours repasser par les intensités (Itotal = ΣI), puis recalculer L.

📐 Atténuation avec la distance
Source ponctuelle (onde sphérique)
$I = \dfrac{P}{4\pi r^2} \quad \Rightarrow\quad I \propto \dfrac{1}{r^2}$

Si la distance double (r → 2r) : I est divisée par 4 → L diminue de 10·log(4) ≈ 6 dB.

📐 Formules clés
Intensité sonore (W/m²)
$I = \dfrac{P}{S}$
Niveau d'intensité sonore (dB)
$L = 10\log\!\left(\dfrac{I}{I_0}\right) \quad I_0 = 10^{-12}\,\text{W·m}^{-2}$
Si I est doublée : L augmente de +3 dB
$L' = L + 3\,\text{dB} \quad (\text{car } 10\log 2 \approx 3)$

✏️ Exercice type BAC
1
Addition de sources sonores
Moyen
Deux haut-parleurs identiques émettent chacun un niveau L = 70 dB. I₀ = 10⁻¹² W·m⁻².
1. Retrouver l'intensité I d'un haut-parleur.
2. Calculer le niveau total Ltot quand les deux sont allumés.
📋 Données
L = 70 dB par haut-parleur · I₀ = 10⁻¹² W·m⁻²
⚠️ Règle fondamentale : les niveaux en dB NE s'additionnent pas ! Il faut passer par les intensités.
① Retrouver I d'un haut-parleur
L = 10·log(I/I₀) → I = I₀ × 10^(L/10)
I = 10⁻¹² × 10^(70/10) = 10⁻¹² × 10⁷ = 10⁻⁵ W·m⁻²
② Additionner les intensités
Itot = I₁ + I₂ = 10⁻⁵ + 10⁻⁵ = 2×10⁻⁵ W·m⁻²
③ Calculer Ltot
Ltot = 10·log(2×10⁻⁵ / 10⁻¹²) = 10·log(2×10⁷)
= 10·(log 2 + log 10⁷) = 10·(0,301 + 7) = 73 dB
💡 Doubler les sources ajoute toujours +3 dB (car 10·log 2 ≈ 3).
✅ 70 + 70 ≠ 140 dB ! Deux sources identiques → L + 3 dB = 73 dB.
