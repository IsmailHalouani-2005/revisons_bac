🍋 Transformation Acide-Base

pH, couples acide/base, transfert de proton H⁺. Le chapitre fondamental de chimie.

🍋 Définitions — Modèle de Brønsted
Acide de Brønsted
Espèce chimique capable de céder un proton H⁺ à une autre espèce. Exemple : CH₃COOH → CH₃COO⁻ + H⁺.
Base de Brønsted
Espèce chimique capable de capter un proton H⁺. Exemple : NH₃ + H⁺ → NH₄⁺.
Couple acide/base conjugués
Acide HA et sa base conjuguée A⁻ liés par : HA ⇌ A⁻ + H⁺. Exemples : CH₃COOH/CH₃COO⁻, NH₄⁺/NH₃, H₂O/HO⁻, H₃O⁺/H₂O.
📊 pH et concentration
Définition du pH
pH=−log[H3​O+](C°=1 mol⋅L−1)
Retrouver [H₃O⁺]
[H3​O+]=10−pH
Produit ionique de l'eau (25°C)
Ke​=[H3​O+][HO−]=10−14pKe​=14
⚠️
Le pKe dépend de la température ! À 25°C → pKe = 14. À 37°C (corps humain) → pKe ≈ 13,6. Utiliser la valeur donnée dans l'énoncé.
⚗️ Réaction acide-base

Une réaction acide-base est un transfert de proton H⁺ de l'acide vers la base. On combine 2 couples :

Méthode des couples
HA1​+A2−​⇌A1−​+HA2​

Sens de la réaction : L'acide le plus fort réagit avec la base la plus forte. L'acide de pKa le plus faible est le plus fort.

🧮 Calcul de pH selon le type d'acide
TypeFormule pHCondition
Acide fort (HCl, HNO₃...)pH = −log CDissociation totale
Base forte (NaOH...)pH = 14 + log CDissociation totale
Acide faiblepH = ½(pKa − log C)Dissociation partielle
Base faiblepH = 14 − ½(pKe − pKa − log C)Dissociation partielle
🧠 L'essentiel en 3 phrases
💡
Acide = donneur de H⁺ (perd un proton). Base = receveur de H⁺ (capte un proton).
Exemple : HCl (acide) donne H⁺ à l'eau. CH₃COO⁻ (base) capte H⁺ de l'eau.

✏️ Exercices type BAC
1
Calcul de pH et concentration
Facile
1. Une solution a pH = 3,5. Calculer [H₃O⁺].
2. Une solution d'acide fort HCl a C = 2×10⁻³ mol/L (totalement dissocié). Calculer son pH.
3. Pour la solution de la question 2, calculer [HO⁻].
📋 Rappels
[H₃O⁺] = 10−pH  |  Ke = [H₃O⁺]·[HO⁻] = 10−14
① pH = 3,5 → [H₃O⁺]
[H₃O⁺] = 10−3,5 = 100,5 × 10−4 = √10 × 10⁻⁴ = 3,16×10⁻⁴ mol·L⁻¹
② HCl fort → dissociation totale → [H₃O⁺] = C
pH = −log(2×10⁻³) = −(log 2 + log 10⁻³) = −(0,301 − 3) = 2,7
💡 log 2 ≈ 0,301 — valeur à connaître par cœur !
③ [HO⁻] = Ke/[H₃O⁺]
[HO⁻] = 10⁻¹⁴ / 2×10⁻³ = 5×10⁻¹² mol·L⁻¹
pH = 2,7 < 7 → solution acide ✓
2
Écrire la réaction acide-base
Facile
Écrire la réaction entre l'acide éthanoïque CH₃COOH et l'ammoniaque NH₃.
Couples : CH₃COOH/CH₃COO⁻ et NH₄⁺/NH₃
📋 Méthode des demi-équations
Couples : CH₃COOH/CH₃COO⁻ (pKa = 4,8) et NH₄⁺/NH₃ (pKa = 9,2)
pKa₁ < pKa₂ → réaction dans le sens direct ✓
① Demi-équation acide (couple 1)
CH₃COOH → CH₃COO⁻ + H⁺
② Demi-équation base (couple 2)
NH₃ + H⁺ → NH₄⁺
③ Addition (H⁺ se simplifie)
CH₃COOH + NH₃ ⇌ CH₃COO⁻ + NH₄⁺
⚠️ Double flèche ⇌ = équilibre
✅ Vérif charges : 0 + 0 = −1 + 1 = 0 ✓
