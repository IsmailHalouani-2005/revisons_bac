📉 Loi de Beer-Lambert

A = k·[X], courbe d'étalonnage, trouver une concentration inconnue par absorbance.

📉 Loi de Beer-Lambert
Énoncé de la loi
Pour une solution colorée diluée, l'absorbance A est proportionnelle à la concentration C de l'espèce absorbante et à l'épaisseur l traversée.
Loi de Beer-Lambert
$A = \varepsilon l C \quad (\varepsilon \text{ en L·mol}^{-1}\text{·cm}^{-1})$
Forme simplifiée (BAC)
$A = kC \quad (k = \varepsilon l = \text{cst})$
⚠️
Conditions de validité : Solution diluée (C ≤ 5×10⁻³ mol/L environ), longueur d'onde fixée à λmax (absorbance maximale), solution homogène.
📊 Méthode de la courbe d'étalonnage
1
Préparer des solutions étalons de concentrations connues.
2
Mesurer l'absorbance A de chaque étalon au spectrophotomètre à λmax.
3
Tracer A = f(C) → droite passant par l'origine si Beer-Lambert vérifié. La pente = k.
4
Mesurer A de la solution inconnue → lire C sur la courbe ou calculer C = A/k.
💡
Si la courbe n'est plus linéaire pour les fortes concentrations → Beer-Lambert n'est plus valable → il faut diluer la solution inconnue avant de mesurer.
🧠 Le principe
💡
Plus une solution est concentrée, plus elle absorbe la lumière. Beer-Lambert dit que c'est une relation linéaire : A = k·[X]. On fait une courbe d'étalonnage (A en fonction de C connues), puis on lit la concentration inconnue sur la courbe.
📐 Formules clés
Loi de Beer-Lambert (1 espèce)
$A = k[X] \quad (k = \varepsilon l)$
Absorbance
$A = \log\!\left(\dfrac{I_0}{I}\right)$
⚠️
Domaine de validité : Beer-Lambert n'est valable que pour des concentrations faibles (généralement [X] ≤ 5×10⁻³ mol·L⁻¹). Au-delà, la courbe n'est plus linéaire (lire sur le graphe !)
📊
Comment utiliser la courbe d'étalonnage :
1) Mesurer A de la solution inconnue avec le spectrophotomètre
2) Lire C sur la courbe à partir de cette valeur de A
Ou calculer k = pente de la droite, puis C = A/k
✏️ Exercice type BAC
1
Courbe d'étalonnage
Moyen
Une courbe A = f(C) est linéaire et passe par (0 ; 0) et (3,5×10⁻³ mol/L ; 1,4).
Une solution inconnue a A = 0,60.
1. Calculer k (pente).
2. Trouver Cinconnue.
📋 Données
Points de la droite : (0 ; 0) et (3,5×10⁻³ mol·L⁻¹ ; 1,4) · Ainconnue = 0,60
① Pente k = ΔA/ΔC
k = (1,4 − 0) / (3,5×10⁻³ − 0) = 1,4 / 3,5×10⁻³ = 400 L·mol⁻¹
💡 Droite passant par l'origine → Beer-Lambert vérifié A = kC
② C = A/k
C = 0,60 / 400 = 1,5×10⁻³ mol·L⁻¹
Vérif : C < 3,5×10⁻³ mol·L⁻¹ → zone linéaire → mesure valide ✓
