🌡️ 1er Principe de la Thermodynamique

Le bilan d'énergie d'un système. Comprendre W, Q et ΔU en 10 minutes.

🧠 L'analogie pour tout comprendre
💡
Imagine un frigo : L'énergie interne U c'est le "stock d'énergie" du système. W c'est le travail (le moteur du frigo travaille). Q c'est la chaleur échangée (le frigo refroidit à l'intérieur = perd de la chaleur).
📐 Les formules à retenir ABSOLUMENT
1er principe général ΔE = ΔEc + ΔEp + ΔU = W + Q
Système au repos (BAC +++) ΔU = W + Q
Système incompressible sans travail ΔU = Q = m·c·ΔT
Gaz parfait p·V = n·R·T
Capacité thermique C = m·c  (en J·K⁻¹)
⚠️
Pièges BAC : W > 0 si le milieu extérieur donne du travail au système. Q > 0 si le système reçoit de l'énergie. T en KELVIN pour les gaz (T = θ°C + 273,15)
🖼️ Schéma bilan énergétique
➡️
W et Q positifs → le système REÇOIT de l'énergie → U augmente
⬅️
W et Q négatifs → le système CÈDE de l'énergie → U diminue
📋 Tableau récapitulatif
SituationFormuleCondition
Cas généralΔE = W + QToujours valable
Au repos (BAC)ΔU = W + QΔEc = ΔEp = 0
Sans travailΔU = Q = mcΔTW = 0 J
Gaz parfaitpV = nRTR = 8,314 J·mol⁻¹·K⁻¹
✏️ Exercices type BAC
1
Chauffage d'un café
Facile
On chauffe 200 g de café (eau, c = 4180 J·kg⁻¹·K⁻¹) de 20°C à 80°C dans un four à micro-ondes (W = 0 J).
1. Calculer ΔT en Kelvin.
2. Calculer la variation d'énergie interne ΔU.
1. ΔT :
ΔT = T_finale − T_initiale = (80+273) − (20+273) = 353 − 293 = 60 K

2. ΔU :
W = 0 donc ΔU = Q = m·c·ΔT
ΔU = 0,200 × 4180 × 60 = 50 160 J ≈ 50,2 kJ

✅ ΔU > 0 : le café reçoit de l'énergie, sa température augmente. Logique !
2
Gaz parfait dans un piston
Moyen
Un gaz parfait occupe un volume V = 2,0 L à T = 300 K et p = 1,0×10⁵ Pa.
1. Calculer la quantité de matière n. (R = 8,314 J·mol⁻¹·K⁻¹)
2. Le gaz reçoit W = −500 J et Q = +800 J. Calculer ΔU.
1. n = pV/RT
n = (1,0×10⁵ × 2,0×10⁻³) / (8,314 × 300)
n = 200 / 2494 = 0,080 mol

2. ΔU = W + Q
ΔU = −500 + 800 = +300 J

✅ Le système gagne 300 J d'énergie interne malgré le travail fourni par le gaz (W négatif), car la chaleur reçue est plus grande.
3
Sujet BAC type — Bilan frigo
Difficile
Un réfrigérateur maintient ses aliments à T₁ = 5°C. Il échange avec l'extérieur à T₂ = 25°C. Son moteur fournit W = −120 J par cycle. Le frigo cède Q₂ = +200 J à la pièce.
1. Quel est le signe de Q₁ (chaleur échangée avec les aliments) ?
2. Calculer Q₁ (1er principe, ΔU = 0 sur un cycle complet).
3. Interpréter le résultat.
1. Signe de Q₁ : Le frigo prélève de la chaleur aux aliments → les aliments cèdent de la chaleur au frigo → Q₁ > 0 pour le frigo (il reçoit).

2. Sur un cycle ΔU = 0 :
ΔU = W + Q₁ + Q₂ = 0
0 = −120 + Q₁ + (−200)   [le frigo cède 200 J à la pièce donc Q₂ = −200 J pour le frigo]
Q₁ = 120 + 200 = +80 J

3. Interprétation : Le frigo prélève 80 J aux aliments, le moteur fournit 120 J de travail, et la somme (200 J) est rejetée dans la pièce. C'est le principe du cycle frigorifique ! ✅

⚙️ 2ème Loi de Newton & Mouvement

Comprendre et appliquer ΣF = ma dans tous les types de mouvement.

🧠 L'analogie simple
💡
C'est simple : Plus tu pousses fort (F grande) et plus l'objet est léger (m petite), plus ça accélère vite. F = ma, c'est tout.
📐 Formules clés
2ème loi de Newton ΣF_ext = m·a
Mouvement rectiligne uniforme (a = 0) v(t) = v₀  |  x(t) = v₀·t + x₀
Mouvement uniformément accéléré a = cst  |  v(t) = a·t + v₀  |  x(t) = ½·a·t² + v₀·t + x₀
Mouvement circulaire uniforme a = v²/R · N̂  (centripète, vers le centre)
⚠️
Règle d'or BAC : Pour passer de l'accélération a(t) à la vitesse v(t), on intègre (primitive). Pour passer de v(t) à x(t), on intègre encore. La constante d'intégration = condition initiale (v₀ ou x₀).
🖼️ Les types de mouvement en un coup d'œil
Typea(t)v(t)x(t)
Rectiligne uniforme0v₀v₀·t + x₀
Rectiligne accéléréa₀ = csta₀·t + v₀½a₀t² + v₀t + x₀
Circulaire uniformev²/R (centripète)v₀ (cst)arc
✏️ Exercices type BAC
1
Voiture freinant
Facile
Une voiture de masse m = 1200 kg roule à v₀ = 20 m/s. Elle freine avec une force F = −6000 N.
1. Calculer l'accélération a.
2. Donner l'équation de v(t).
3. Quand s'arrête-t-elle ?
1. ΣF = ma → a = F/m = −6000/1200 = −5 m·s⁻²

2. v(t) = a·t + v₀ = −5t + 20

3. v(t) = 0 → −5t + 20 = 0 → t = 20/5 = 4 s
2
Satellite en orbite circulaire
Moyen
Un satellite de masse m = 500 kg orbite à R = 7000 km du centre de la Terre. g = 8,2 m·s⁻².
1. Identifier la seule force appliquée.
2. Appliquer Newton pour trouver v (vitesse orbitale).
1. Seule force : poids P = mg (attraction gravitationnelle vers la Terre).

2. Newton : P = m·a_centripète
mg = m·v²/R
v² = g·R = 8,2 × 7×10⁶ = 5,74×10⁷
v = √(5,74×10⁷) ≈ 7 577 m·s⁻¹ ≈ 7,6 km/s

🪂 Mouvement dans le Champ de Pesanteur

Chute libre, tir parabolique, portée et flèche. Le chapitre le plus demandé au BAC.

🧠 L'image mentale
💡
Lance un ballon en diagonal : horizontalement, il va à vitesse constante (pas de friction). Verticalement, la gravité le tire vers le bas. Ces deux mouvements sont INDÉPENDANTS. La trajectoire résultante est une parabole.
📐 Équations du mouvement (à connaître par cœur)
Accélération (seule force = poids) aₓ = 0     aᵧ = −g
Vitesse (primitive de a) vₓ(t) = v₀·cos α     vᵧ(t) = −g·t + v₀·sin α
Position (primitive de v) x(t) = v₀·cos α · t + x₀     y(t) = −½g·t² + v₀·sin α · t + y₀
Équation de la trajectoire (y en fonction de x) y(x) = −g·x² / (2·(v₀·cos α)²) + x·tan α
⚠️
Si lancé horizontalement (α = 0) : sin 0 = 0 et cos 0 = 1, donc vₓ = v₀, vᵧ = −gt, x = v₀t, y = −½gt²
🖼️ Schéma du tir parabolique
QuantitéFormule
Flèche (alt. max)y_max = v₀²·sin²α / (2g)
Temps à la flèchet_flèche = v₀·sin α / g
PortéeChercher t tel que y(t) = 0, puis x(t)
✏️ Exercices type BAC
1
Bille lancée horizontalement
Facile
Une bille est lancée horizontalement depuis une table (hauteur h = 1,25 m) avec v₀ = 3 m/s. g = 10 m/s².
1. Écrire x(t) et y(t) (origine = bord de la table).
2. Quand touche-t-elle le sol ?
3. Quelle est la portée horizontale ?
1. α = 0 → x(t) = 3t  |  y(t) = −½×10×t² + 0 = −5t²
(y₀ = 0 car on prend le bord de la table comme origine, le sol est à y = −1,25 m)

2. y(t) = −1,25 m → −5t² = −1,25 → t² = 0,25 → t = 0,5 s

3. x = 3 × 0,5 = 1,5 m
2
Tir oblique (BAC classique)
Moyen
Un ballon est lancé avec v₀ = 15 m/s à α = 30° depuis le sol. g = 10 m/s².
1. Exprimer x(t) et y(t). (cos 30° ≈ 0,866 ; sin 30° = 0,5)
2. Calculer la hauteur maximale.
3. Calculer la portée (quand y revient à 0).
1. x(t) = 15×0,866×t = 13t  |  y(t) = −5t² + 15×0,5×t = −5t² + 7,5t

2. Hauteur max : vᵧ = 0 → −10t + 7,5 = 0 → t = 0,75 s
y_max = −5×(0,75)² + 7,5×0,75 = −2,81 + 5,63 = 2,81 m

3. Portée : y = 0 → t(−5t + 7,5) = 0 → t = 1,5 s
x = 13 × 1,5 = 19,5 m

⚡ Mouvement dans un Champ Électrique Uniforme

Condensateur plan, déviation de particule chargée. Analogue du tir parabolique.

🧠 L'analogie
💡
C'est exactement pareil que la pesanteur ! Sauf que la "gravité" est remplacée par la force électrique. La particule fait aussi une parabole. La seule différence : a = qE/m (pas g).
📐 Formules clés
Champ électrique dans un condensateur E = U_AB / d    (V·m⁻¹)
Force sur une charge q F_e = q·E = m·a  →  a = qE/m
Équations du mouvement (entrée horizontale) aₓ=0, aᵧ=qE/m  |  vₓ=v₀, vᵧ=aᵧ·t  |  x=v₀t, y=½aᵧt²
Conservation énergie mécanique (accélérateur) ½mv²_B = q·U_AB  →  v_B = √(2qU_AB/m)
⚠️
Attention au signe ! Si q est négatif (électron), la force est dans le sens opposé à E. Le champ E est toujours orienté du + vers le −.
✏️ Exercice type BAC
1
Déviation d'un électron dans un condensateur
Moyen
Un électron (q = −1,6×10⁻¹⁹ C, m = 9,1×10⁻³¹ kg) entre horizontalement avec v₀ = 5×10⁷ m/s dans un condensateur. U_AB = 200 V, d = 4 cm.
1. Calculer E.
2. Calculer |a| de l'électron.
3. Pour une longueur de condensateur L = 5 cm, calculer la déviation verticale.
1. E = U/d = 200 / 0,04 = 5000 V·m⁻¹

2. a = |q|E/m = (1,6×10⁻¹⁹ × 5000) / (9,1×10⁻³¹) = 8×10⁻¹⁶ / 9,1×10⁻³¹ ≈ 8,8×10¹⁴ m·s⁻²

3. Temps dans le condo : t = L/v₀ = 0,05 / (5×10⁷) = 10⁻⁹ s
y = ½at² = ½ × 8,8×10¹⁴ × (10⁻⁹)² = 4,4×10⁻⁴ m = 0,44 mm

🪐 Satellites & Lois de Kepler

Les 3 lois qui régissent le mouvement de toutes les planètes et satellites.

📐 Les 3 lois de Kepler
1️⃣
Loi des orbites : Chaque planète décrit une ellipse dont le Soleil est un foyer. (Cercle = cas particulier)
2️⃣
Loi des aires : Le satellite balaie des aires égales en des temps égaux → il va plus vite au périhélie (plus près) et plus lentement à l'aphélie.
3️⃣
Loi des périodes : T²/a³ = k (constante dépendant uniquement de l'astre attracteur)
3ème loi de Kepler T² / R³ = 4π² / (G·M_T)
Vitesse satellite orbite circulaire v = √(G·M_T / R)    ou    v = 2πR/T
⚠️
G = 6,67×10⁻¹¹ N·m²·kg⁻² | M_Terre = 5,97×10²⁴ kg | R en mètres, T en secondes !
✏️ Exercices type BAC
1
Satellite géostationnaire
Moyen
Un satellite géostationnaire tourne avec la Terre (T = 24h). G·M_T = 4,0×10¹⁴ m³·s⁻².
1. Convertir T en secondes.
2. Utiliser la 3ème loi de Kepler pour trouver R.
1. T = 24×3600 = 86 400 s

2. T²/R³ = 4π²/(G·M_T)
R³ = G·M_T·T²/(4π²) = 4×10¹⁴ × (86400)² / (4π²)
R³ = 4×10¹⁴ × 7,46×10⁹ / 39,5 = 7,55×10²²
R = (7,55×10²²)^(1/3) ≈ 4,22×10⁷ m = 42 200 km
2
Comparaison de deux planètes
Facile
La Terre a T_T = 1 an et a_T = 1 UA. Mars a a_M = 1,52 UA.
Calculer la période de révolution de Mars.
T²/a³ = cst (même astre attracteur = Soleil)
T_M²/a_M³ = T_T²/a_T³
T_M² = T_T² × (a_M/a_T)³ = 1² × (1,52)³ = 3,51
T_M = √3,51 ≈ 1,87 ans

🔊 Intensité Sonore

Décibels, niveau d'intensité sonore, atténuation. La physique du son.

📐 Formules clés
Intensité sonore (W/m²) I = P / S
Niveau d'intensité sonore (dB) L = 10·log(I / I₀)    I₀ = 10⁻¹² W·m⁻²
Si I est doublée : L augmente de +3 dB L' = L + 3 dB   (car 10·log(2) ≈ 3)
⚠️
PIÈGE BAC : Les niveaux sonores NE s'additionnent PAS. Deux sources à 60 dB = 63 dB (pas 120 dB). Les INTENSITÉS s'additionnent : I_tot = ΣI.
SituationNiveau (dB)
Seuil d'audibilité0 dB
Conversation60–70 dB
Seuil de risque85 dB
Seuil de douleur120 dB
✏️ Exercice type BAC
1
Addition de sources sonores
Moyen
Deux haut-parleurs identiques émettent chacun un niveau L = 70 dB. I₀ = 10⁻¹² W·m⁻².
1. Retrouver l'intensité I d'un haut-parleur.
2. Calculer le niveau total L_tot quand les deux sont allumés.
1. L = 10·log(I/I₀) → I = I₀ × 10^(L/10) = 10⁻¹² × 10^7 = 10⁻⁵ W·m⁻²

2. I_tot = 2I = 2×10⁻⁵ W·m⁻²
L_tot = 10·log(2×10⁻⁵ / 10⁻¹²) = 10·log(2×10⁷) = 10·(log 2 + 7) = 10·(0,301+7) = 73 dB

💧 Dynamique des Fluides

Bernoulli, débit, Archimède. La physique des liquides en mouvement.

📐 Formules clés
Poussée d'Archimède Π = ρ_fluide · V_immergé · g
Débit volumique (conservation) D_V = S·v = constante  →  S₁·v₁ = S₂·v₂
Théorème de Bernoulli p + ½ρv² + ρgh = constante
💡
Bernoulli en simple : Si le tuyau se rétrécit (S diminue), v augmente et p diminue. C'est l'effet Venturi (aspirateur, carburateur).
✏️ Exercice type BAC
1
Tuyau avec étranglement
Moyen
Un fluide (ρ = 1000 kg/m³) coule dans un tuyau horizontal. En A : S_A = 4 cm², v_A = 2 m/s, p_A = 1,5×10⁵ Pa. En B : S_B = 1 cm².
1. Calculer v_B (conservation du débit).
2. Calculer p_B (Bernoulli, même hauteur h_A = h_B).
1. S_A·v_A = S_B·v_B → v_B = (S_A/S_B)·v_A = (4/1)×2 = 8 m·s⁻¹

2. p_A + ½ρv_A² = p_B + ½ρv_B² (h cst)
p_B = p_A + ½ρ(v_A² − v_B²) = 1,5×10⁵ + ½×1000×(4−64)
p_B = 1,5×10⁵ − 30 000 = 1,2×10⁵ Pa
✅ p diminue quand v augmente (effet Venturi).

🔦 Diffraction

Déviation de la lumière par une fente. Formule θ = λ/a et tache centrale.

🧠 L'image mentale
💡
Imagine une vague qui passe par une ouverture : si l'ouverture est petite par rapport à la longueur d'onde, elle s'étale après (comme une vague de mer passant par un port). C'est la diffraction !
📐 Formules clés
Écart angulaire θ = λ / a    (rad)
Largeur tache centrale (approximation petits angles) θ ≈ L/(2D)  →  L = 2·λ·D/a
Relation inversée λ = a·L/(2D)    ou    a = 2·λ·D/L
⚠️
Unités : λ en mètres (1 nm = 10⁻⁹ m, 1 μm = 10⁻⁶ m). a = taille de la fente, L = largeur tache centrale, D = distance fente-écran.
💡
Règle intuitive : Plus la fente est petite, plus la tache est large. Plus la longueur d'onde est grande (rouge), plus la tache est large.
✏️ Exercice type BAC
1
Mesure de la taille d'une fente
Facile
Un laser (λ = 633 nm) éclaire une fente. L'écran est à D = 1,5 m. On mesure une tache centrale de largeur L = 3,8 cm.
Calculer la taille de la fente a.
a = 2·λ·D / L = (2 × 633×10⁻⁹ × 1,5) / (3,8×10⁻²)
a = 1,9×10⁻⁶ / 3,8×10⁻² = 5,0×10⁻⁵ m = 50 μm

🌈 Interférences

Fentes de Young, interfrange, franges brillantes et sombres.

🧠 L'image mentale
💡
Deux cailloux dans l'eau : les cercles se croisent. Là où les crêtes se rencontrent = interférence constructive (frange brillante). Là où crête + creux = interférence destructive (frange sombre).
📐 Formules clés
Interfrange (distance entre 2 franges brillantes) i = λ·D / a
Différence de marche δ = S₂M − S₁M = a·x / D
Frange brillante (constructive) δ = k·λ   (k entier)
Frange sombre (destructive) δ = (k + ½)·λ
⚠️
i vs a : i = interfrange (ce qu'on mesure sur l'écran), a = distance entre les 2 fentes (petite), D = distance fentes-écran (grande). λ en mètres !
✏️ Exercice type BAC
1
Fentes de Young classique
Facile
Deux fentes séparées de a = 0,4 mm sont éclairées par λ = 589 nm. L'écran est à D = 1,2 m.
1. Calculer l'interfrange i.
2. On mesure 11 franges sur 4,3 cm. Vérifier i.
1. i = λD/a = (589×10⁻⁹ × 1,2) / (0,4×10⁻³) = 7,07×10⁻⁷ / 4×10⁻⁴ = 1,77×10⁻³ m ≈ 1,8 mm

2. Pour 11 franges, on a 10 interfranges :
i = 4,3 cm / 10 = 4,3 mm...
Recalcul : i = 43 mm / 10 = 4,3 mm → cohérent si D = 3 m (vérifier l'énoncé). La méthode est : i = mesure totale / nombre d'espaces.

🚑 Effet Doppler

Décalage de fréquence quand émetteur et récepteur se déplacent l'un vers l'autre.

🧠 L'image mentale
💡
L'ambulance : Quand elle s'approche → son plus aigu (f plus grande). Quand elle s'éloigne → son plus grave (f plus petite). Pour la lumière : s'approche = blueshift (décalage vers le bleu), s'éloigne = redshift (décalage vers le rouge).
📐 Formules clés
Rappel : vitesse d'onde v_onde = λ · f
Émetteur se rapproche (f reçue > f émise) f_A = f_émetteur × c / (c − v_émetteur)
Émetteur s'éloigne (f reçue < f émise) f_B = f_émetteur × c / (c + v_émetteur)
Décalage Doppler Δf = f_reçue − f_émise
⚠️
c = vitesse du son (340 m/s dans l'air) ou vitesse de la lumière (3×10⁸ m/s) selon le contexte.
✏️ Exercice type BAC
1
Radar de vitesse (Doppler)
Moyen
Un radar émet une onde à f₀ = 24 GHz (c = 3×10⁸ m/s). Une voiture s'approche à 50 m/s. Calculer la fréquence reçue par le radar.
f_A = f₀ × c/(c − v) = 24×10⁹ × 3×10⁸/(3×10⁸ − 50)
f_A = 24×10⁹ × 3×10⁸/2,9999995×10⁸
f_A ≈ 24×10⁹ × 1,0000000167 ≈ 24 000 000 004 Hz
Δf = 4 Hz (décalage très faible car v << c, c'est normal pour la lumière/radio)

🔌 Circuit RC

Charge et décharge d'un condensateur, constante de temps τ = RC, équation différentielle.

🧠 L'image mentale
💡
Le condensateur = une éponge électrique : Il se remplit (charge) puis se vide (décharge). Plus R est grand, plus ça prend du temps. τ = RC = le temps pour remplir 63% de l'éponge.
📐 Formules clés
Constante de temps τ = R · C    (secondes)
Tension charge (condensateur vide → E) u_C(t) = E · (1 − e^(−t/τ))
Tension décharge (condensateur plein → 0) u_C(t) = E · e^(−t/τ)
Équation différentielle (charge) du_C/dt + u_C/(RC) = E/(RC)
⚠️
À t = τ : charge → u_C = 0,63E | décharge → u_C = 0,37E. À t = 5τ : condensateur considéré plein ou vide (≈ 99%).
🖼️ Graphes charge / décharge
✏️ Exercice type BAC
1
Charge d'un condensateur
Moyen
Un circuit RC avec R = 10 kΩ, C = 100 μF, E = 12 V. Le condensateur est initialement déchargé.
1. Calculer τ.
2. Donner l'expression de u_C(t).
3. Quelle est la tension à t = τ ?
4. À partir de quand le condensateur est-il "chargé" ?
1. τ = RC = 10×10³ × 100×10⁻⁶ = 1 s

2. u_C(t) = E(1 − e^(−t/τ)) = 12(1 − e^(−t))

3. à t = τ = 1s : u_C = 12(1 − e⁻¹) = 12×0,63 = 7,6 V ≈ 0,63E

4. "Chargé" à t = 5τ = 5 s (u_C ≈ 99% de E)

Moyen

🍋 Transformation Acide-Base

pH, couples acide/base, transfert de proton H⁺. Le chapitre fondamental de chimie.

🧠 L'essentiel en 3 phrases
💡
Acide = donneur de H⁺ (perd un proton). Base = receveur de H⁺ (capte un proton).
Exemple : HCl (acide) donne H⁺ à l'eau. CH₃COO⁻ (base) capte H⁺ de l'eau.
📐 Formules clés
pH d'une solution pH = −log[H₃O⁺]    et    [H₃O⁺] = 10^(−pH)
Produit ionique de l'eau (25°C) Ke = [H₃O⁺]·[HO⁻] = 10⁻¹⁴    pKe = 14
Réaction acide-base générale HA₁ + A₂⁻ ⇌ A₁⁻ + HA₂
⚠️
pH 7 = neutre ([H₃O⁺] = [HO⁻])
pH < 7 → acide ([H₃O⁺] > [HO⁻])
pH > 7 → basique ([H₃O⁺] < [HO⁻])
SolutionpH[H₃O⁺]
Acide fort (HCl, 0,1 mol/L)10,1 mol/L
Eau pure (25°C)710⁻⁷ mol/L
Base forte (NaOH, 0,1 mol/L)1310⁻¹³ mol/L
✏️ Exercices type BAC
1
Calcul de pH et concentration
Facile
1. Une solution a pH = 3,5. Calculer [H₃O⁺].
2. Une solution d'acide fort HCl a C = 2×10⁻³ mol/L (totalement dissocié). Calculer son pH.
3. Pour la solution de la question 2, calculer [HO⁻].
1. [H₃O⁺] = 10^(−3,5) = 3,16×10⁻⁴ mol·L⁻¹

2. HCl fort → [H₃O⁺] = C = 2×10⁻³ mol/L
pH = −log(2×10⁻³) = −(log2 + log10⁻³) = −(0,3 − 3) = 2,7

3. Ke = [H₃O⁺][HO⁻] → [HO⁻] = 10⁻¹⁴ / 2×10⁻³ = 5×10⁻¹² mol·L⁻¹
2
Écrire la réaction acide-base
Facile
Écrire la réaction entre l'acide éthanoïque CH₃COOH et l'ammoniaque NH₃.
Couples : CH₃COOH/CH₃COO⁻ et NH₄⁺/NH₃
Méthode : l'acide du couple 1 réagit avec la base du couple 2.
Acide 1 : CH₃COOH → CH₃COO⁻ + H⁺
Base 2 : NH₃ + H⁺ → NH₄⁺

Bilan : CH₃COOH + NH₃ ⇌ CH₃COO⁻ + NH₄⁺ ✅

⚖️ Force des Acides et des Bases

Ka, pKa, acides forts vs faibles, classement. Solution tampon.

🧠 La règle simple
💡
pKa petit = acide FORT (il donne facilement H⁺). pKa grand = acide faible (il garde son H⁺). HCl a pKa = −7 (très fort). L'acide acétique a pKa = 4,8 (faible).
📐 Formules clés
Constante d'acidité Ka = [A⁻]·[H₃O⁺] / [HA]    pKa = −log Ka
Relation pKa et pH (Henderson-Hasselbalch) pH = pKa + log([A⁻]/[HA])
À pH = pKa [HA] = [A⁻]   (moitié dissocié)
⚠️
Diagramme de prédominance : Si pH < pKa → HA prédomine. Si pH > pKa → A⁻ prédomine.
AcidepKaForce
H₃O⁺0Acide fort
CH₃COOH (acétique)4,8Faible
NH₄⁺9,2Très faible
H₂O14Extrêmement faible
✏️ Exercice type BAC
1
Diagramme de prédominance
Facile
pKa(CH₃COOH/CH₃COO⁻) = 4,8. Pour une solution de pH = 6 :
1. Quelle espèce prédomine ?
2. Calculer [CH₃COO⁻]/[CH₃COOH].
1. pH = 6 > pKa = 4,8 → la base CH₃COO⁻ prédomine.

2. pH = pKa + log([A⁻]/[HA])
6 = 4,8 + log([A⁻]/[HA])
log([A⁻]/[HA]) = 1,2
[CH₃COO⁻]/[CH₃COOH] = 10^1,2 ≈ 15,8
✅ Il y a environ 16× plus de base que d'acide.

🎨 Titrage Coloriмétrique

Trouver la concentration d'une solution par changement de couleur à l'équivalence.

🧠 Le principe en 1 image
💡
On verse goutte à goutte la solution titrante (connue) dans la solution titrée (inconnue). Quand la solution change de couleur → on a ajouté exactement la bonne quantité → c'est l'équivalence. On lit V_E sur la burette.
📐 Formules clés
À l'équivalence (proportions stœchiométriques) n_A / a = n_B / b  →  C_A·V_A/a = C_B·V_E/b
Concentration de la solution titrée C_A = (a × C_B × V_E) / (b × V_A)
⚠️
a et b = coefficients stœchiométriques de l'équation. V_E = volume équivalent (lu sur la burette). V_A et V_E en même unité (mL ou L) !
IndicateurZone de virage (pH)Acide→Base
Rouge de crésol7,4–9,0Jaune → Violet
Bleu de bromothymol6,0–7,6Jaune → Bleu
Phénolphtaléine8,0–9,6Incolore → Rose
✏️ Exercice type BAC
1
Titrage HCl par NaOH
Facile
V_A = 20,0 mL de HCl de concentration inconnue. On titre avec NaOH à C_B = 0,050 mol/L. L'indicateur change de couleur pour V_E = 12,5 mL.
Équation : HCl + NaOH → NaCl + H₂O (a = b = 1)
Calculer C_A(HCl).
C_A = (a × C_B × V_E) / (b × V_A)
C_A = (1 × 0,050 × 12,5) / (1 × 20,0)
C_A = 0,625 / 20,0 = 0,031 mol·L⁻¹

📊 Titrage Conductimétrique

L'équivalence est repérée par un changement de pente sur la courbe de conductivité.

🧠 Le principe
💡
On mesure la conductivité σ de la solution au fur et à mesure qu'on ajoute le titrant. Avant l'équivalence et après, les ions présents changent → la conductivité varie différemment → rupture de pente = équivalence !
📐 Formules clés
À l'équivalence (même formule) C_A = (a × C_B × V_E) / (b × V_A)
💡
Comment trouver V_E sur la courbe ? Tracer les 2 droites (avant et après rupture). L'intersection = V_E.
Ionλ (mS·m²·mol⁻¹)
H₃O⁺35,01 (très conducteur)
HO⁻19,9
Na⁺5,01
Cl⁻7,63
⚠️
Explication de la rupture : Titrage HCl par NaOH. Avant V_E : on enlève H₃O⁺ (très conducteur) et on ajoute Na⁺ (peu conducteur) → σ baisse. Après V_E : on ajoute Na⁺ + HO⁻ → σ monte.
✏️ Exercice
1
Lecture graphique V_E
Facile
Sur une courbe conductimétrique, les deux droites se croisent à V_E = 11 mL. C_B(NaOH) = 0,050 mol/L, V_A = 20 mL.
Calculer C_A(HCl). (a = b = 1)
C_A = C_B × V_E / V_A = 0,050 × 11 / 20 = 0,0275 mol·L⁻¹

📈 Titrage pH-métrique

Courbe pH = f(V), saut de pH à l'équivalence. Méthode des tangentes parallèles.

📐 Formules + méthodes
À l'équivalence (toujours) C_A = (a × C_B × V_E) / (b × V_A)
📏
Méthode des tangentes parallèles : Tracer 2 tangentes parallèles à la courbe de part et d'autre du saut de pH. Tracer une droite parallèle à mi-chemin entre les deux. Son intersection avec la courbe = V_E.
📏
Méthode de la dérivée : dpH/dV = maximum au point d'équivalence. V_E correspond au pic de la dérivée.
⚠️
À la demi-équivalence (V = V_E/2) : pH = pKa. C'est une méthode pour déterminer le pKa de l'acide titré !
✏️ Exercice type BAC
1
Titrage pH-métrique complet
Moyen
Titrage de V_A = 20 mL d'acide faible HA par NaOH (C_B = 0,10 mol/L). Sur la courbe : V_E = 9,3 mL, pH à V_E/2 = 4,65 mL → pH = 4,8.
1. Calculer C_A.
2. Donner le pKa du couple HA/A⁻.
1. C_A = C_B × V_E / V_A = 0,10 × 9,3 / 20 = 0,0465 mol·L⁻¹

2. À la demi-équivalence, pH = pKa → pKa = 4,8

🔬 Spectroscopie UV-Vis & IR

Identifier des molécules par leur spectre d'absorption. Cercle chromatique, bandes IR.

🧠 Les 2 types de spectroscopie
☀️

🌡️

💡

📉 Loi de Beer-Lambert

A = k·[X], courbe d'étalonnage, trouver une concentration inconnue par absorbance.

🧠 Le principe
💡
Plus une solution est concentrée, plus elle absorbe la lumière. Beer-Lambert dit que c'est une relation linéaire : A = k·[X]. On fait une courbe d'étalonnage (A en fonction de C connues), puis on lit la concentration inconnue sur la courbe.
📐 Formules clés
Loi de Beer-Lambert (1 espèce) A = k · [X]    avec k = ε·l (coefficient × épaisseur)
Absorbance A = log(I₀/I)    (sans unité)
⚠️
Domaine de validité : Beer-Lambert n'est valable que pour des concentrations faibles (généralement [X] ≤ 5×10⁻³ mol·L⁻¹). Au-delà, la courbe n'est plus linéaire (lire sur le graphe !)
📊
Comment utiliser la courbe d'étalonnage :
1) Mesurer A de la solution inconnue avec le spectrophotomètre
2) Lire C sur la courbe à partir de cette valeur de A
Ou calculer k = pente de la droite, puis C = A/k
✏️ Exercice type BAC
1
Courbe d'étalonnage
Moyen
Une courbe A = f(C) est linéaire et passe par (0 ; 0) et (3,5×10⁻³ mol/L ; 1,4).
Une solution inconnue a A = 0,60.
1. Calculer k (pente).
2. Trouver C_inconnue.
1. k = ΔA/ΔC = (1,4 − 0) / (3,5×10⁻³ − 0) = 1,4 / 3,5×10⁻³ = 400 L·mol⁻¹

2. C = A/k = 0,60 / 400 = 1,5×10⁻³ mol·L⁻¹

⚡ Conductimétrie

σ = k·C, loi de Kohlrausch. Mesurer une concentration par la conductivité.

📐 Formules clés
Loi de Kohlrausch (solution diluée) σ = Σ λ_i · [X_i]    (S·m⁻¹)
Avec un seul soluté ionique σ = k · C    (linéaire !)
💡
Même principe que Beer-Lambert : σ est proportionnel à C. On fait une courbe d'étalonnage σ = f(C), et on lit C pour une valeur de σ inconnue.
⚠️
Conversion : 1 mol·m⁻³ = 10⁻³ mol·L⁻¹. Attention aux unités des conductivités molaires (S·m²·mol⁻¹ ou mS·cm⁻¹ selon le contexte) !
✏️ Exercice
1
Lecture courbe conductimétrique
Facile
Une courbe σ = f(C) est une droite de pente k = 8,5 mS·cm⁻¹/(10⁻³ mol·L⁻¹). Une solution inconnue a σ = 0,022 mS·cm⁻¹.
Calculer C.
σ = k·C → C = σ/k = 0,022 / 8,5 = 2,6×10⁻³ mol·L⁻¹

⏱️ Cinétique Chimique

Vitesse de réaction, ordre 1, temps de demi-réaction t₁/₂. Lent ou rapide ?

📐 Formules clés
Vitesse volumique de disparition du réactif v_disp(R) = −d[R]/dt
Loi vitesse ordre 1 v(A) = k · [A]    (k = constante de vitesse en s⁻¹)
Concentration loi ordre 1 [A]_t = [A]₀ · e^(−k·t)
Temps de demi-réaction t₁/₂ = ln2 / k ≈ 0,693 / k
💡
Reconnaître l'ordre 1 au BAC : Si ln[A] = f(t) est une droite → ordre 1. Si t₁/₂ est indépendant de [A]₀ → ordre 1.
⚠️
Facteurs cinétiques : ↑ T → réaction plus rapide. ↑ concentration → réaction plus rapide. Catalyseur → accélère sans être consommé.
✏️ Exercice type BAC
1
Ordre 1 — calcul de t₁/₂
Moyen
Une réaction suit une loi d'ordre 1 avec k = 2,3×10⁻² s⁻¹. [A]₀ = 0,10 mol/L.
1. Calculer t₁/₂.
2. Calculer [A] à t = 60 s.
1. t₁/₂ = ln2/k = 0,693 / 2,3×10⁻² = 30 s

2. [A] = [A]₀·e^(−kt) = 0,10 × e^(−2,3×10⁻² × 60) = 0,10 × e^(−1,38) = 0,10 × 0,25 = 0,025 mol·L⁻¹

⚖️ Constante d'Équilibre

K, Qr, sens d'évolution, taux d'avancement τ. Réaction totale ou équilibrée ?

📐 Formules clés
Quotient de réaction (aA + bB ⇌ cC + dD) Q_r = ([C]/C°)^c · ([D]/C°)^d / (([A]/C°)^a · ([B]/C°)^b)   C° = 1 mol/L
À l'équilibre Q_r,éq = K
Sens d'évolution Q_r < K → sens direct | Q_r > K → sens indirect
Taux d'avancement τ = x_f / x_max    τ = 1 → totale | τ < 1 → équilibrée
💡
L'eau et les solides n'apparaissent PAS dans Q_r ! (activité = 1)
✏️ Exercice type BAC
1
Tableau d'avancement + taux τ
Moyen
CH₃COOH + H₂O ⇌ CH₃COO⁻ + H₃O⁺
n₀(CH₃COOH) = 0,10 mol, n₀(H₂O) = solvant.
À l'équilibre, x_f = 1,3×10⁻³ mol.
1. Calculer x_max.
2. Calculer τ et conclure.
1. x_max = min(n₀/coeff) = min(0,10/1) = 0,10 mol
(L'eau est en solvant = excès, donc le réactif limitant est CH₃COOH)

2. τ = x_f/x_max = 1,3×10⁻³ / 0,10 = 0,013 = 1,3%
τ << 1 → la réaction est très peu avancée → acide faible (peu dissocié). ✅

🔧 Mécanismes Réactionnels

Sites donneurs/accepteurs d'électrons, flèches courbes. La chimie au niveau microscopique.

🧠 Les règles d'or
💡
Flèche courbe = déplacement de doublet d'électrons : La flèche va du doublet (donneur) vers l'atome qui l'accepte. Jamais l'inverse !
Sites DONNEURS (nucléophiles)Sites ACCEPTEURS (électrophiles)
Doublet non liant (N, O, ...)Lacune électronique (H⁺, Li⁺, ...)
Liaison multiple C=C, C=OLiaison polarisée C^δ⁺-Cl^δ⁻ (côté δ⁺)
Liaison polarisée côté δ⁻Atome chargé positivement
✏️ Exercice type BAC
1
Identifier les sites
Moyen
Dans la réaction : HO⁻ + CH₃-Cl → CH₃-OH + Cl⁻
1. Identifier HO⁻ : site donneur ou accepteur ?
2. Identifier le carbone dans CH₃-Cl : donneur ou accepteur ?
3. Dessiner la flèche courbe de l'attaque.
1. HO⁻ a des doublets non liants → donneur (nucléophile)

2. C-Cl est polarisée (Cl est très électronégatif) → C est δ⁺ → accepteur (électrophile)

3. La flèche va de HO⁻ (doublet O) vers le C^δ⁺ de CH₃Cl. Une 2ème flèche part de la liaison C-Cl vers Cl (Cl repart avec les 2 électrons). ✅

⚡ Électrolyse

Transformation forcée d'énergie électrique en chimique. Calcul de masse déposée.

🧠 Anode vs Cathode — le moyen mnémotechnique
💡
OA CA = "Oh! A Ca !" → Oxydation à l'Anode, Réduction à la Cathode. Pareil pour les piles !
📐 Formules clés
Charge électrique totale Q = I · Δt    Q = n(e⁻) · F
Nombre de moles d'électrons n(e⁻) = I · Δt / F    F = 96 500 C·mol⁻¹
Masse déposée (électrodépôt) m = n · M = (I · Δt · M) / (z · F)    z = nb d'e⁻ échangés
⚠️
z = nombre d'électrons échangés dans la demi-équation. Ex: Cu²⁺ + 2e⁻ → Cu : z = 2. Ag⁺ + e⁻ → Ag : z = 1.
✏️ Exercice type BAC
1
Électrodépôt de cuivre
Moyen
Électrolyse d'une solution de CuSO₄. I = 2,0 A, Δt = 30 min. M(Cu) = 63,5 g/mol, F = 96 500 C/mol.
Demi-équation cathode : Cu²⁺ + 2e⁻ → Cu (z = 2)
Calculer la masse de cuivre déposée.
Q = I·Δt = 2,0 × (30×60) = 2,0 × 1800 = 3600 C
n(e⁻) = Q/F = 3600/96500 = 0,0373 mol
n(Cu) = n(e⁻)/z = 0,0373/2 = 0,0187 mol
m(Cu) = n × M = 0,0187 × 63,5 = 1,19 g

🔋 Piles Électrochimiques

Transfert spontané d'électrons, oxydoréduction, calcul de durée de vie.

🧠 Pile vs Électrolyse
PILEÉLECTROLYSE
RéactionSpontanéeForcée
ÉnergieChimique → ÉlectriqueÉlectrique → Chimique
AnodePôle − (oxydation)Pôle + (oxydation)
CathodePôle + (réduction)Pôle − (réduction)
📐 Formules clés
Capacité maximale Q_max = n(e⁻) · F
Durée de fonctionnement Δt = Q_max / I = n(e⁻) · F / I
💡
Méthode : 1) Écrire les 2 demi-équations. 2) Trouver n(réactif limitant). 3) Déduire n(e⁻) via stœchiométrie. 4) Q = n(e⁻)·F. 5) Δt = Q/I.
✏️ Exercice type BAC
1
Pile Zn/Cu — durée de vie
Moyen
Pile Zn/Cu. Anode : Zn → Zn²⁺ + 2e⁻. Cathode : Cu²⁺ + 2e⁻ → Cu.
n(Zn) = 0,050 mol est le réactif limitant. I = 0,50 A. F = 96 500 C/mol.
Calculer la durée de fonctionnement.
n(e⁻) = 2 × n(Zn) = 2 × 0,050 = 0,10 mol
Q = n(e⁻)·F = 0,10 × 96500 = 9650 C
Δt = Q/I = 9650/0,50 = 19 300 s ≈ 5,4 h

☢️ Transformations Nucléaires

Radioactivité α β γ, loi de désintégration, demi-vie t₁/₂. Loi de Soddy.

📐 Les 4 types de désintégration
TypeParticule émiseVariation AVariation Z

Loi de désintégration N(t) = N₀ · e^(−λt)    A(t) = A₀ · e^(−λt)
Demi-vie t₁/₂ = ln2 / λ ≈ 0,693 / λ
⚠️
Loi de Soddy : Conservation de A (nombre de masse) et Z (numéro atomique) dans chaque désintégration. Écrire l'équation avec ΣA et ΣZ conservés des deux côtés !
✏️ Exercices type BAC
1
Équation de désintégration α
Facile
Le Radium 226 (²²⁶₈₈Ra) émet une particule α. Écrire l'équation de désintégration et identifier le noyau fils.
²²⁶₈₈Ra → ᴬ_Z X + ⁴₂He
Conservation A : 226 = A + 4 → A = 222
Conservation Z : 88 = Z + 2 → Z = 86
Z = 86 = Radon (Rn)
✅ ²²⁶₈₈Ra → ²²²₈₆Rn + ⁴₂He
2
Calcul avec la demi-vie
Moyen
Un isotope a t₁/₂ = 5730 ans (carbone 14). N₀ = 1,0×10¹² noyaux. Calculer N après 11 460 ans.
11 460 = 2 × 5730 = 2 × t₁/₂ → 2 demi-vies écoulées
Après 1 demi-vie : N = N₀/2 = 5,0×10¹¹
Après 2 demi-vies : N = N₀/4 = 2,5×10¹¹ noyaux

Vérification : N = N₀·e^(−λt) avec λ = ln2/5730
N = 10¹² × e^(−ln2 × 2) = 10¹² × (1/4) = 2,5×10¹¹ ✅
